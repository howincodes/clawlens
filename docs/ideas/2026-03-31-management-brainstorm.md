# Management Brainstorm — 2026-03-31

Raw ideas from management meeting. Nothing finalized yet.

## 1. Task Management System
- Use ClawLens as the team's task management system
- Create projects, assign users to projects
- Collect project repository names
- Collect each person's GitHub ID

## 2. Attendance Management + Salary Calculation
- Track attendance via prompt activity (did a person work today?)
- Salary calculation module tied to attendance/activity data

## 3. Git History Analysis
- Analyze each developer's git history
- Correlate git activity with prompt activity
- Know what they worked on and when

## 4. Client Application — Out of Stealth
- Make the client-side component visible (not hidden/silent)
- Currently the hook script is invisible to the developer
- Move to a well-visible, branded client presence

## 5. Full Remote Config Control
- Fully control Claude Code and Codex configs from the server
- Push config changes remotely (not just hooks — full settings)

## 6. Improved Activity History Tracking
- Current tracking has gaps or delays
- Move toward real-time or near-real-time activity tracking
- Better "antigravity" (anti-gravity = anti-idle/anti-ghost) detection

## 7. Open for More Ideas
- Team welcomes additional feature suggestions
